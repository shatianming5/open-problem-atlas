# OpenProblemAtlas 宣言

## 我们要解决的问题

许多重要的开放问题至今未解，不仅是因为它们难，还因为：

- **分散** — 散落在论文、问题集、社区页面和专家维护站点中
- **难以发现** — 没有深厚的领域知识就找不到
- **难以比较** — 缺乏标准化元数据
- **缺少可操作入口** — 对 AI 工具、定理证明器和计算搜索不友好

## 我们的使命

构建一个**公共基础设施**，使开放问题可搜索、可验证、可贡献、可追踪——面向研究者、学生、AI 智能体和定理证明器。

## 核心原则

### 1. 精度优先于覆盖度
宁可有 300 个完美验证的问题，也不要 10,000 条噪声记录。

### 2. 分层架构
三个不同层级防止污染：Atlas（零噪声）、Radar（允许噪声）、Lab（明确标注）。

### 3. 来源可追溯
每个事实必须追溯到规范来源。机器生成内容有明确标记，永远不混入真相层。

### 4. 开放治理
状态变更（尤其是 open → solved）需要证据和人工审核。

### 5. AI 是工具，不是预言者
LLM 帮助提取、总结和候选发现。它们永远不能单方面决定问题是否开放、规范陈述是什么、或证明是否有效。

## v1 范围

**领域：** 数学、理论 CS、数学物理（精确/可形式化的子领域）

**v1 不做：** 生物、化学、广义物理谜题、需要大量实验背景的问题
