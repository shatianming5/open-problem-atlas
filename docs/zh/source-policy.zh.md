# 来源策略

## 来源分级

### 一级：规范来源
可作为已验证问题的主要参考。
- 问题集（如 Clay 千禧年问题、Erdős 问题）
- 原始论文
- 综述和教科书
- 专家维护页面
- 形式化猜想仓库

### 二级：辅助来源
可佐证状态或提供上下文，但不能作为已验证条目的唯一来源。
- 社区讨论（MathOverflow 等高质量回答）
- 基准论文
- 权威数据库（OEIS、MathWorld、nLab）

### 三级：仅限线索来源
只能进入雷达层。永远不能作为已验证条目的唯一依据。
- 新闻文章
- 博客帖子
- 社交媒体
- 视频内容

## 规则

1. 每个已验证问题必须引用至少一个一级来源作为规范陈述依据。
2. 每个状态声明必须引用至少一个来源作为证据。
3. 三级来源只能在雷达层创建线索。
4. 禁止来源伪造。
5. 来源元数据必须包含：类型、标题、作者、年份、URL 或 DOI、定位符。

## 版权与合理使用

- 永远不要将全文 PDF 或完整文章提交到仓库
- 仅存储：元数据、定位符、短摘录（合理使用）和自己的结构化注释
- 遵守所有程序化访问的 API 服务条款
