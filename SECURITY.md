# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public issue.**

Instead, email: security@openproblem-atlas.org (or contact the maintainer directly)

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

## Scope

This project is primarily a data repository with supporting tooling. Security concerns may include:

- Injection vulnerabilities in data processing pipelines
- Supply chain risks in dependencies
- Credential exposure in configuration files
- XSS or injection in the static site generator
